# 递归行为和递归行为的时间复杂度分析

递归方法取最大值
base case 只剩一个数返回；否则左部分最大值，右部分最大值，比较得到总的最大值

取中点的方法
(A+B)/2  当A,B很大时可能会溢出
A+(B-A)/2
右移一位等同于除2(而且比除法更快)
A+((B-A) >> 1)

分析递归画决策树

master公式——递归函数的时间复杂度
T(N) = T(N/2) + T(T/2) + O(1)
T(N) = 2T(N/2) + O(1)

master公式模型
T(N) = aT(N/b)+ O(N^d)
适用条件：子问题等规模
1. log b a < d => O(N^d)
2. log b a > d => O(N^log b a)
3. log b a = d => O(N^d * logN)
